# Java OOP Practice Solutions

A collection of Java Object-Oriented Programming (OOP) practice problems and lab assignments developed for the **SWE 06132124 – Object Oriented Programming Language Lab** course.

## Topics Covered

- Classes and Objects
- Constructors and Constructor Overloading
- Method Overloading and Method Overriding
- Encapsulation
- Inheritance
- Polymorphism
- Arrays and 2D Arrays
- String Manipulation
- Loops and Conditional Statements
- Static Members
- Abstract Classes
- Basic System Design

## Getting Started

### Clone the Repository

```bash
git clone git@github.com:BumbleBee-100/Task.git
```

### Navigate to the Project

```bash
cd Task
```

### Run the Programs

1. Open the project in IntelliJ IDEA.
2. Navigate to the desired practice problem folder.
3. Compile and run the corresponding Java file.
